## 3. 주제 — "AI로 [ ]을/를 없앤다면?"

전국 80개 대학, 500여 개의 팀이 단 하나의 질문에 답합니다.
**"AI로 ( )를 없앨 수 있다면, 그 세상은 어떻게 달라질까요?"**

- 빈칸에 들어갈 문제는 각 팀이 직접 정의·검증한다.
- 개발이 필수가 아니며, 아이디어를 구체화한 **프로토타입 또는 기획안 제작**을 목표로 한다.
- 멋사대학 아이디어톤에서 그 관점을 끝까지 시험해 보고, 진짜 MAKER로 한 걸음 더 나아가 보세요.

**예시 키워드(공식 제안)**: 사회적 비교 / 심리적 작용 / 정보의 불균형 / 자원 사각지대 / ETC / **"우리 팀의 주제는?"**
**공식 시안 표현**: "AI로 **가짜 뉴스** 을/를 없앤다면?"

## 5. 참여 방법 (HOW TO JOIN)

### 01. 팀 모집 (TEAM)
- 팀은 **최소 2인, 최대 6인**으로 구성 가능. 모든 팀원은 **동일한 대학 소속의 멋사 14기**여야 함.
- **팀장**이 커뮤니티 플랫폼에서 팀원 모집 글을 작성, 팀원들은 해당 팀 참가를 신청.
- 모든 팀원이 참가한 후 **팀장이 모집 마감**해야 **제출 페이지가 오픈**됨.
- 모집 마감 이후 **팀원 변경 불가**, 한 명당 **하나의 팀**에만 소속 가능.

## 9. 우리(weave) 기획에 직결되는 시사점

| 대회 사실 | 우리 기획에 미치는 의미 |
|---|---|
| 주제 = **"AI로 [ ]를 없앤다면?"** | **AI가 없애는 대상**이 한 줄로 떨어져야 함 → REVIEW_NOTES.md A1 그대로 적중 |
| **개발 가산점**(작동만 판단, 1회) + AI 바이브 코딩 환영 | 풀 프로덕트 아닐 필요 없음. **핵심 기능 1개 배포**로 가산 확보. MCP 폐기/단순화 결정과 정합 |
| **시장 반응도 = 외부인 포함 좋아요 20%** | **카톡 공유·바이럴 설계**가 곧 점수. 외부 친구가 1탭으로 '좋아요' 누를 수 있어야 함 → A5(카카오) 결정 강력히 정당화 |
| **100초 피칭 + 500자 설명 + 5분 IR DECK** | 우리 서비스가 **이 산출물 자체를 자동 생성**해 주면 메타 매력 폭발 (AI 서비스로 AI 공모전 산출물 만들기) |
| **1차 100팀 → 2차 10팀 → 본선 10팀** | 1차 통과(상위 100/2244팀 ≈ 상위 5%)가 첫 관문. **'영상 임팩트 + 좋아요'**가 1차 핵심 |
| 심사 기준 100% 분배: 문제정의/차별성/시장반응도 각 20% | "AI 활용 깊이"는 **차별성 20%** 항목. A2 깊이 확보 필요 |
| 발표자 포함 **최소 1인 오프라인 본선** | 팀 안에 **현장 참석 가능한 발표자** 1명 확정해두기 |
| 멋사 14기 + **동일 대학** | 우리 팀이 이 조건 만족하는지 확인 필요 (멋사 14기? 동일 대학?) |
| 1차 마감까지 **카운트다운 ~7일대** | 일정이 매우 타이트. **최소 기능 → 좋아요 모으기 → 영상**까지 우선순위 재정렬 필요 |
